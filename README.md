## Run Locally

Clone the project

```bash
  git clone https://github.com/Haider-Mukhtar/lenob-project.git
```

Go to the project directory

```bash
  cd lenob-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
